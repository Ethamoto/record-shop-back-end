\# Project Name



ASP.NET Core Web API project built with .NET 8.



\## Tech Stack



\- C#

\- ASP.NET Core Web API

\- Entity Framework Core

\- SQL Server

\- NUnit for testing

\- Swagger / OpenAPI



\## How to Run



1\. Clone the repo

2\. Open the solution in Visual Studio

3\. Restore NuGet packages

4\. Update the database connection string if needed

5\. Run the project
